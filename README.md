# Login Application

A simple login application built using **React (Vite)** and **Node.js + Express**.

## Features

- Login with username and password
- Backend API validation
- Error handling
- Navigation to welcome page
- Username stored in localStorage

## Tech Stack

Frontend: React, Vite  
Backend: Node.js, Express

### Frontend

cd Frontend
